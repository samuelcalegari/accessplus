# Accessibility Block for Moodle #

## Current version ##

alpha-0.1.0

## Features ##
- Theme selection
- increase and decrease font

## Supported languages ##
- english
- french

## Installation ##

copy file into directory /blocks/accesplus/ and visit /admin/index.php in your browser

## Requirement ##

Moodle 3.1 or greater.

## Other infos ##

Use Jquery and ES6 Features
